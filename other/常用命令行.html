# vpn

默认用的 Cisco 的客户端，代理地址是 127.0.0.1:9999

开启后 npm 出了点问题，这里给 npm 单独设置过代理，如果以后出问题的话可以删除这个配置

npm config set proxy 127.0.0.1:9999
npm config set https-proxy 127.0.0.1:9999

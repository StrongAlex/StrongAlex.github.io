# Vuepress配置笔记

# 结构以及说明

文章写在docs文件夹下面，写好文章后记得配置侧边栏

项目主要的配置在docs\.vuepress\config.js里面，注意.vuepress这个文件夹是默认隐藏的

docs下的README.md是配置首页

deploy.sh是发布所需的脚本

package.json是一些基本配置，比如配置运行发布脚本

# 启动发布项目

启动项目

npm run docs:dev
1

发布到线上

npm run deploy
